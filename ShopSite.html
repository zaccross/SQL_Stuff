Connecting to Sk8Shoppe using MySQL/PHP

Final Project for CIS 551 For Zac Cross :)

Below you will find the 10 applications to access my Sk8Shoppe Database, and then the .txt files that created them.

APPLICATION 1: Search for items by Brand
Please enter a brand name to get a list of all related products. Current brands include: ['Real', 'Girl', and 'Alien Workshop']


APPLICATION 2: Search for contest winners by contest
Please enter a contest code to get a list of the winners and their prizes. Current contest with winners only include: ['1', '2', '4']


APPLICATION 3: View your order total by looking up your order number
Please enter an order number to view the total price after discount. Current order numbers only include: ['1', '2',..., '10']


APPLICATION 4: View your favorite team rider's set up
Please enter the last name of a team rider. Examples include ['Scoots','Scoops', 'Os'].


APPLICATION 5: Recommendation System
Please enter a customer number for a customer specific recommendation.
Examples include ['1','2',...,'1O'].


APPLICATION 6: Making a Customer Account
Please enter your information in the following order:
A Unique Customer Number
First Name
Last Name
Email
And Address (if living in Boring Town)






APPLICATION 7: CHECK for Making a Customer Account
Press Submit to view all customer account numbers and names to see if app 6 is successful

APPLICATION 8: Search for a Team Rider's Promotional Content
Please enter a Rider's First and Last name to get a list of videos and ads they appear in:. Current riders include: ['Ryder Scoots', 'Deandra Scon']


APPLICATION 9: Cancel and item on an order / remove and item from an order
Please enter a valid Customer and Item number to delete the data:.



APPLICATION 10: CHECK FOR APPLICATION 9: Cancel and item on an order / remove and item from an order
Prints out list of customer nums and products ordered to check delete function of app 9.
